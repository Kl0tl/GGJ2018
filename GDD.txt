JoueurManette/BlindPlayer :
	3C : Character 	: 	Se déplace dans le maze
						Entends les sons du maze
						Peut ping sa position sur la map
		Camera 		: none
		Controls 	: XBOX PAD : Arrow UP : Avancer
								 Arrow DOWN : Reculer
								 Arrow RIGHT : Tourner de 90° sur la droite
								 Arrow LEFT : Tourner de 90° sur la gauche
								 LT : Strafe Gauche
								 RT : Strafe Droit

JoueurClavier/MuteDeafPlayer : 
	3C : Character 	:	Voir la carte du maze
						Créer un son de ping dans le maze
						Annoter sur la carte
		Camera		:	Carte du maze
		Controls	:	Souris

Carte
La carte représente la totalité du maze.
Le triangle sur la carte est le BlindPlayer.
La sortie n'est pas visible sur la carte.

Maze
La sortie du labyrinthe émets un son.


Sounds : 
	Sortie : exit.wav
	Ping JoueurClavier : comehere.wav
